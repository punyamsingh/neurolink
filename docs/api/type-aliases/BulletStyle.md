Bullet style options for formatting

- "disc": Standard bullet point (•)
- "number": Numbered list (1. 2. 3.)
- "checkmark": Checkmark symbol (✓)
- "arrow": Arrow symbol (→)
- "dash": Dash symbol (–)
- "none": No bullet marker
